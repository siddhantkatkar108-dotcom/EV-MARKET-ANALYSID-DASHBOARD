# EV Market Analysis Dashboard

## Project Overview
This project analyzes Electric Vehicle (EV) sales data using Excel, SQL, and Power BI. The dashboard provides insights into EV market trends, sales performance, vehicle categories, and regional distribution.

## Objectives
- Analyze EV sales performance.
- Identify top-performing vehicle categories.
- Track revenue and sales trends.
- Compare regional EV adoption.
- Generate actionable business insights.

## Tools Used
- Microsoft Excel
- SQL
- Power BI

## Dataset Features
- Vehicle Category
- Manufacturer
- Region
- Sales Quantity
- Revenue
- Price Category
- Year
- Market Share

## Dashboard KPIs
- Total Sales
- Total Revenue
- Average Revenue
- Top Vehicle Category
- Regional Performance
- Sales Trend Analysis

## Process
1. Data Cleaning in Excel
2. Data Transformation
3. SQL Analysis
4. Power BI Data Modeling
5. DAX Measure Creation
6. Dashboard Development
7. Insight Generation

## Key Insights
- Identified top-performing EV categories.
- Analyzed revenue contribution by region.
- Evaluated market growth trends.
- Compared sales performance across categories.
